# 🛍️ Home - Clothing Store

Home - Clothing Store is a modern and responsive front-end web project designed to showcase a stylish fashion e-commerce layout. Built using **HTML** and **CSS**, this project highlights clean UI design, well-structured layouts, and mobile responsiveness. The website is deployed on **Vercel** for fast and reliable hosting.

---

## 🚀 Live Demo  
🔗 **Hosted on Vercel:** *[https://online-shopping-web-zeta.vercel.app/]*

---

## 📌 Features

- ✔️ Modern and minimalistic UI  
- ✔️ Responsive design for mobile, tablet & desktop  
- ✔️ Clean navigation bar  
- ✔️ Product showcase and collections  
- ✔️ Promotional banners / hero section  
- ✔️ Pure HTML & CSS (no frameworks)  

---

## 🛠️ Technologies Used

- **HTML5** – Semantic page structure  
- **CSS3** – Custom styling, layout, responsiveness  
- **Vercel** – Hosting & deployment

🚀 Deployment on Vercel
Option 1: Using Vercel Dashboard

1.  Go to: https://vercel.com/dashboard

2.  Import your GitHub repository

3.  Click Deploy


Option 2: Using Vercel CLI

Install the Vercel CLI : npm install -g vercel
Deploy : vercel
